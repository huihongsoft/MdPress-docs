界面概览

Tags: 指南 界面

MdPress 的界面设计简洁直观，类似于 VS Code，让开发者感到亲切。

1. 活动栏 (Activity Bar)

位于界面最左侧，包含主要的功能模块入口：

  • 资源管理器 (Explorer): 管理当前工作区的文件和文件夹。
  • 搜索 (Search): 在当前工作区进行全局文本搜索。
  • 站点管理 (Sites): 专门用于管理 MdPress 静态站点项目，配置站点信息。
  • 扩展 (Extensions): 管理已安装的插件，如导出工具。
  • 设置 (Settings): 访问软件设置。

2. 侧边栏 (Sidebar)

点击活动栏图标后显示的面板，展示具体内容。

  • 文件树: 显示项目目录结构，支持右键菜单操作（新建、删除、重命名）。
  • 插件面板: 显示已安装和推荐的插件，可在此进行启用/禁用操作。

3. 编辑器区域 (Editor Area)

占据界面主体的核心区域。

  • Monaco 编辑器: 支持语法高亮、代码折叠、多光标编辑。
  • 工具栏: 顶部提供常用 Markdown 格式化快捷按钮（加粗、斜体、列表、链接、图片、表格等）。
  • 面包屑导航: 显示当前文件的路径。

4. 预览区域 (Preview Area)

默认在编辑器右侧（可配置），实时渲染 Markdown 内容。

  • 支持同步滚动：编辑区域滚动时，预览区域自动跟随。
  • 支持点击跳转：在预览区点击标题，编辑器自动定位到对应源码位置。

5. 状态栏 (Status Bar)

位于界面底部，显示当前状态信息：

  • 语言模式: 当前文件的语言（如 Markdown）。
  • 光标位置: 行号和列号。
  • 通知信息: 显示构建成功、导出完成等系统通知。

MdPress 简介

Tags: 指南 介绍

MdPress 是一款基于 Electron 构建的现代化 Markdown 编辑器与静态站点生成器。它结合了 VS Code 级别的编辑体验与灵活的建站功能，致力于为开发者、技术写作者和知识管理爱好者提供从创作到发布的一站式解决方案。

核心特性

1. 专业级编辑器

  • Monaco Editor 内核: 采用与 VS Code 相同的编辑器核心，提供代码高亮、智能提示和极速响应。
  • 实时预览: 所见即所得的 Markdown 渲染，支持 GFM (GitHub Flavored Markdown) 标准。
  • 数学公式: 内置 KaTeX 支持，完美渲染数学公式。
  • 拆分视图: 支持编辑器与预览窗口的左右拆分，提升写作效率。

2. 强大的文件管理

  • 工作区模式: 支持打开文件夹作为工作区，轻松管理包含数百个文档的大型项目。
  • 文件树: 直观的侧边栏文件资源管理器，支持新建、重命名、移动和删除操作。

3. 静态站点生成 (SSG)

  • 一键构建: 内置静态站点生成引擎，无需配置复杂的 Node.js 环境，即可将 Markdown 文档转化为高性能的静态网站。
  • 多主题支持: 内置多款精美主题（Official, Default, Basic），满足不同场景需求。
  • SEO 友好: 自动生成语义化的 HTML 结构，有利于搜索引擎收录。

4. 插件与扩展

  • 多格式导出: 通过插件系统，支持将文档导出为 单文件 HTML、PDF、Word、EPUB 和 CHM 工程。
  • 扩展市场: (开发中) 未来将支持从市场下载更多主题和功能插件。

5. 安全与隐私

  • 本地优先: 所有数据存储在本地，不依赖云端服务，保障您的数据隐私。
  • 沙箱隔离: 插件运行在隔离环境中，确保软件运行安全。

技术栈

MdPress 站在巨人的肩膀上：

  • Electron: 跨平台桌面应用框架。
  • React: 构建流畅的用户界面。
  • Vite: 极速的开发与构建工具。
  • Monaco Editor: 业界领先的代码编辑器组件。

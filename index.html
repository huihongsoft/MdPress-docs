贴合团队的文档工程工作流

从本地编辑到站点发布，每一步都围绕开发者习惯与 CI/CD 能力设计，帮助你把文档当作代码管理。

  1. 在本地通过 Monaco 编辑器编辑 Markdown 文档，使用熟悉的快捷键与命令面板。
  2. 通过实时预览确认渲染效果，包含数学公式、图表与代码高亮。
  3. 使用工作区管理不同站点，按项目组织文档目录结构。
  4. 将内容提交到 Git 仓库，由 CI/CD 自动构建并部署静态站点。

为什么选择 MdPress

如果你正在为团队寻找一款既像编辑器又像站点构建器的工具，MdPress 将是合适的选择。

面向工程团队

遵循开发者习惯的界面与交互方式，让写文档像写代码一样自然，降低团队采纳成本。

专注文档而非通用 IDE

不是通用代码编辑器，而是围绕文档结构化、链接关系与导航体验做了深度优化。

可视化站点结果

在本地即可感知最终站点的布局与导航结构，减少上线后返工调整的成本。

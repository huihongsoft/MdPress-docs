# 构建静态网站

完成内容创作后，下一步就是将 Markdown 文档转化为可发布的静态 HTML 网站。

## 执行构建

1. 在 MdPress 中打开您的站点项目。
2. 点击侧边栏的 **Sites** 面板。
3. 在当前站点卡片上，点击 **Build**（构建）按钮（通常是锤子或播放图标）。

## 构建过程

构建过程中，MdPress 会执行以下操作：
1. **解析配置**: 读取 `config.json`。
2. **处理资源**: 复制 `assets` 目录下的静态资源。
3. **渲染 Markdown**: 将所有 `.md` 文件转换为 HTML，应用主题模板。
4. **生成索引**: 创建搜索索引和分类/标签页面。

## 构建产物

构建完成后，在项目根目录下会生成一个名为 `dist` (或 `build`) 的文件夹。

```text
dist/
├── assets/
├── Guide/
│   ├── intro.html
│   └── ...
├── index.html
└── 404.html
```

这个文件夹包含了网站运行所需的所有文件，您可以将其直接部署到任何静态 Web 服务器。

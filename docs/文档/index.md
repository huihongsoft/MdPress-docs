# 简介

欢迎阅读 MdPress 指南！

MdPress 是一个专为轻松创建文档站点而设计的静态站点生成器。

## 特性

### 以 Markdown 为中心
使用 Markdown 编写您的内容。

### React 驱动
在 Markdown 中使用 React 组件。

### 极速体验
基于 Vite 构建，提供即时的服务器启动体验。

查看 [安装](/documents/guides/installation.html) 指南以开始使用。

### 使用

# 快速开始 (Quick Start)

本指南将带您快速上手 MdPress，从创建项目到预览生成的网站。

## 1. 新建站点项目

1. 打开 MdPress，点击左侧活动栏的 **Sites** 图标。
2. 点击 **New Site** 按钮。
3. 选择一个空文件夹作为项目根目录。
4. MdPress 会自动为您生成基础的项目结构，包含 `docs/` 目录和 `config.json` 配置文件。

## 2. 编写您的第一篇文档

1. 在左侧资源管理器中，展开 `docs` 目录。
2. 点击 **新建文件** 图标，创建一个名为 `hello.md` 的文件。
3. 输入以下内容：

   ```markdown
   ---
   title: 你好，MdPress
   date: 2023-10-01
   ---

   # 欢迎使用 MdPress

   这是我的第一篇文档。MdPress 让写作变得如此简单！

   ## 列表测试
   - 简单
   - 高效
   - 强大
   ```

## 3. 实时预览

在您输入的同时，右侧的预览窗口会实时更新渲染效果。您可以随时检查格式是否正确。

## 4. 导出与构建

- **导出单文件**: 点击右上角的 **导出** 图标（或在插件面板使用 Markdown Exporter），选择 **Export Single HTML**，即可生成一个独立的 HTML 文件分享给他人。
- **构建整站**: 点击 **Sites** 面板中的 **Build** 按钮。构建完成后，`dist` 目录中将生成完整的静态网站。
- **本地预览**: 构建成功后，点击 **Open Preview** 按钮，将在浏览器中打开生成的网站。

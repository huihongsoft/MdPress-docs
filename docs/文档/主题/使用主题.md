# 使用主题

MdPress 采用数据与展示分离的设计，通过切换主题，您可以瞬间改变站点的外观。

## 内置主题

目前 MdPress 提供了以下官方主题：

1. **Official (官方主题)**: 专为技术文档设计，包含侧边栏、导航栏和目录索引，风格简洁现代。
2. **Default (默认主题)**: 基础的博客/文档风格。
3. **Basic (极简主题)**: 仅包含最核心的内容渲染，适合需要高度自定义的用户。

## 切换主题

打开 `config.json` 文件，修改 `theme` 字段：

```json
{
  "theme": "official"
}
```

或者：

```json
{
  "theme": "default"
}
```

保存配置文件后，重新构建站点即可看到效果。

# Frontmatter Configuration

Reference for frontmatter options.

## Fields

### title
Defines the page title.

### slug
Defines the custom URL slug for the page.

### description
Defines the page description for SEO.

### layout
Defines the layout type (doc or home).

## Advanced

### head
Additional head tags.

### outline
Configure the outline sidebar level.
